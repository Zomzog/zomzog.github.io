The contract can be directly imported in swagger-ui

With docker: docker run -p 80:8080 -e URLS="[{url: \"https://gist.githubusercontent.com/Zomzog/6b2f0b59a93e176cd6e71cde393a91e3/raw/598decdf460683baea0db14ad46ed026192c0fff/contract.yaml\", name: \"Ponies\"}]" swaggerapi/swagger-ui

Generate documentation with openapi-generator

Asciidoc exemple: npx @openapitools/openapi-generator-cli generate -i openapi.yaml -g asciidoc -o ./asciidoc

A custom template can be easly created that will allow to generate constant documentation format for all APIs.
